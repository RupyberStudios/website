Forge
  1. Download the Forge Loader
  2. If a dependency download GeckoLib
  3. Download the Forge version of the mod that you want to install
  4. Execute the Forge Installer
  5. Open the .minecraft folder:
    • Windows: use Ctrl+R and type %appdata%, then go into the .minecraft/ folder
    • Linux: open your Home folder and go to .minecraft/ (the folder is hidden)
  6. Open the mods/ folder and move there the .jar files
  7. Launch the Forge installation from the Laucher
